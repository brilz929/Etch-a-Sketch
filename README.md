# Etch-a-Sketch

A modern, web-based implementation of the classic Etch-a-Sketch toy built with vanilla JavaScript, HTML, and CSS.

## Features

- **Interactive Drawing**: Hover over grid squares to create colorful artwork
- **Random Colors**: Each stroke generates a random color for creative variety
- **Adjustable Grid Size**: Create grids from 1x1 to 100x100 squares
- **Reset Function**: Clear the canvas with one click while preserving grid size
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Smooth Animations**: Includes hover effects and transitions for better UX

## How to Use

1. **Drawing**: Move your mouse over the white grid squares to draw with random colors
2. **New Grid**: Click "New Grid" to create a custom-sized grid (1-100 squares per side)
3. **Reset**: Click the reset button (↻) to clear all drawings while keeping the current grid size

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with Flexbox, responsive design, and animations
- **Vanilla JavaScript**: Interactive functionality and DOM manipulation
- **Google Fonts**: Caveat font for a playful, handwritten aesthetic
- **Font Awesome**: Icons for UI elements

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start creating your digital artwork!
